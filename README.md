### 4. "왜 무료인가" 흐름 (신뢰·정당성)
`취약계층 무상 교육 → 정부 교육 지원금 수령 → 학부모 교육비로 환원`
- 목적: "왜 공짜로 돈을 주지?" 의심 해소 → 전환
- 마무리: *일회성 이벤트가 아니라, 학부모와 교육기관이 함께 성장하는 교육 상생 시스템*

- **블루 섹션 배경** = `linear-gradient(#4385F2→#2C66EC)` + radial 광원 5곳 + SVG 그레인(opacity .25). (지원금·최종 CTA 공통)

### 타이포 / 형태
- 폰트 **Pretendard** (Regular 400 / Medium 500 / SemiBold 600 / Bold 700), 강조는 굵기로만(세리프 X)
- `word-break: keep-all` (한글 단어 중간 안 끊김)
- Radius: 버튼·pill 999 / 카드 20
- 히어로 H1 clamp(40~72) / 섹션 제목 clamp(26~40) / 금액 34~52

### 톤앤매너
- **신뢰 · 따뜻 · 정직** — 이벤트성 과장/세일 배너 인상 금지
- 블루 절제(포인트만), 숫자 크고 또렷
- UX writing 하이브리드: 가벼운 피드백=해요체 / 결정·경고=합니다체
- **금지**: 컬러 이모지 / 궁서체 / 세리프 숫자 / 과한 인터랙션(글로우 보더·둥둥 뜨는 효과)
